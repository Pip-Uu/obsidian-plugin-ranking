🌟持续在榜专题推荐

连续上榜 ≥4 周 📌月榜 / ≥12 周 🏅季榜

🏅 Dataview — 知识库查询引擎

  • 在榜位置：全品类 #1（铁打不动）
  • 持续霸榜原因：把 Markdown 笔记库变成 SQL 可查询数据库，几乎所有高级工作流都依赖它
  • 实战用法：TABLE FROM "项目" WHERE status = "进行中" SORT priority DESC
  • 配合插件：Tasks、Calendar、Kanban 均可结合 Dataview 展示

🏅 Excalidraw — 手绘可视化

  • 在榜位置：全品类 #2 / 图像类 #3
  • 持续在榜原因：Obsidian 生态中无可替代的绘图工具，流程图/架构图/思维导图一统
  • 进阶技巧：LaTeX 公式可直接嵌入绘图；![[文件名.excalidraw]] 嵌入笔记

🏅 Tasks — 任务管理

  • 在榜位置：全品类 #3
  • 持续在榜原因：Obsidian 最成熟的任务系统，支持截止日期/重复/优先级/查询
  • 核心语法：- [ ] 完成报告 📅 2026-06-30 ⏫ 🔁 每周五

🏅 Templater — 模板增强

  • 在榜位置：全品类 #4
  • 持续在榜原因：比核心模板插件强大数倍，JS 脚本+日期变量实现自动化

📌 Advanced Slides — 幻灯片之王

  • 在榜位置：全品类 #10 / PPT类 #1
  • 入榜周数：8 周
  • 专题点评：Obsidian 演示场景的绝对主力。基于 reveal.js，网格布局+演讲者视图+双向联动三大杀手锏。20 页 PPT 制作时间从 2 小时压缩到 30 分钟。
  • 推荐配置：theme: moon + highlightTheme: monokai + 开启演讲者视图

📌 Omnisearch + Text Extractor — 搜索组合拳

  • 在榜位置：全品类 #7
  • 入榜周数：6 周
  • 专题点评：搭配 Text Extractor 可搜索 PDF/图片中的文字，搭配 AI Image Analyzer 可语义搜索图片内容。配置一次，终身受益。

📌 Smart Connections — AI 对话知识库

  • 在榜位置：全品类 #9
  • 入榜周数：5 周
  • 专题点评：向知识库提问"有哪些关于注意力机制的笔记？"，AI 基于语义理解返回结果，比关键词搜索精准得多。AI 类插件中热度上升最快的。
