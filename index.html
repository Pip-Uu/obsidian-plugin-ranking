🌟持续在榜专题推荐

连续上榜 ≥4 周 📌月榜 / ≥12 周 🏅季榜

🏅 Excalidraw — 手绘可视化

  • 在榜位置：全品类 #1（新晋榜首）/ 图像类 #1
  • 为什么持续在榜：6.5M 下载量远超第二，Obsidian 生态中无可替代的绘图工具
  • 进阶技巧：LaTeX 公式可直接嵌入绘图；![[文件名.excalidraw]] 嵌入笔记
  • 相关笔记：[[30.areas/finance/Investment lessons/2024.Current trading lessons.md]]

🏅 Dataview — 知识库查询引擎

  • 在榜位置：全品类 #2（从#1微降）
  • 为什么持续在榜：9.1K GitHub Stars 仍是所有插件最高，几乎所有高级工作流都依赖它
  • 实战用法：TABLE FROM "项目" WHERE status = "进行中" SORT priority DESC
  • 配合插件：Tasks、Calendar、Kanban 均可结合 Dataview 展示
  • ⚠️ 关注：Obsidian 官方 Bases 功能逐步推进，长期可能分流

🏅 Tasks — 任务管理

  • 在榜位置：全品类 #3（稳如磐石）
  • 为什么持续在榜：Obsidian 最成熟的任务系统，支持截止日期/重复/优先级/查询
  • 核心语法：- [ ] 完成报告 📅 2026-06-30 ⏫ 🔁 每周五

🏅 Templater — 模板增强

  • 在榜位置：全品类 #4（4.8M 下载量，模板天花板）
  • 为什么持续在榜：比核心模板插件强大数倍，JS 脚本+日期变量实现自动化

📌 Advanced Slides — 幻灯片之王

  • 在榜位置：PPT类 #1（跌出全品类Top10但分类仍霸榜）
  • 入榜周数：9 周
  • 专题点评：Obsidian 演示场景的绝对主力。基于 reveal.js，网格布局+演讲者视图+双向联动三大杀手锏。20 页 PPT 制作时间从 2 小时压缩到 30 分钟。
  • 推荐配置：theme: moon + highlightTheme: monokai + 开启演讲者视图

📌 Omnisearch + Text Extractor — 搜索组合拳

  • 在榜位置：全品类 #10
  • 入榜周数：7 周
  • 专题点评：搭配 Text Extractor 可搜索 PDF/图片中的文字，搭配 AI Image Analyzer 可语义搜索图片内容。配置一次，终身受益。

📌 Smart Connections — AI 对话知识库

  • 在榜位置：全品类 #9
  • 入榜周数：6 周
  • 专题点评：向知识库提问"有哪些关于注意力机制的笔记？"，AI 基于语义理解返回结果，比关键词搜索精准得多。AI 类插件中热度上升最快的。

📌 Obsidian Git — 版本控制

  • 在榜位置：全品类 #5（本期跃升3位）
  • 入榜周数：6 周
  • 专题点评：11.5K GitHub Stars 全插件最高，自动备份+版本历史+多设备同步。开发者群体的标配插件，近期因多设备协作需求持续升温。

🆕 Copilot — AI 编程助手 🆕新上榜

  • 在榜位置：全品类 #7（🆕首次上榜）
  • 入榜周数：1 周
  • 专题点评：7.3K Stars / 1.5M 下载，AI 类插件中热度极高。直接在 Obsidian 中调用 AI 对话、改写、总结内容，与 Smart Connections 互补——Smart Connections 偏语义搜索，Copilot 偏对话交互。

🆕 Canvas Presentation — Canvas 演示 🆕新上榜

  • 在榜位置：PPT类 #4（🆕首次上榜）
  • 入榜周数：1 周
  • 专题点评：基于 Obsidian 原生 Canvas 画布做演示，区别于传统 slide-by-slide 模式，更适合非线性展示。与 Advanced Canvas 插件配合效果更佳。
